# SpecDrop

**SpecDrop** is an AI-powered project management utility built during the **#EveryoneShipsNow** for World Product Day. It bridges the gap between raw product thoughts and actionable development pipelines by instantly transforming Product Requirement Documents (PRDs) into structured GitHub issues.

## Features

* **Instant PRD Parsing:** Input messy, conversational feature descriptions or full PRDs and let AI handle the heavy lifting.
* **Structured Ticket Generation:** Automatically extracts User Stories, Acceptance Criteria, and priority tags.
* **Direct GitHub Integration:** Push generated issues straight to your target repository with a single click.
* **Telemetry Equipped:** Full behavioral analytics mapping powered natively by Novus.ai.

## Tech Stack

* **Frontend/Sandbox:** [Bolt.new](https://bolt.new)
* **AI Engine:** Google Gemini API
* **Deployment:** Bolt Host / Vercel
* **Analytics & Code Mapping:** Novus.ai

## Installation & Setup

1. Open the deployed application link.
2. Provide your custom **Gemini API Key** and **GitHub Personal Access Token** directly in the UI.
3. Paste your product specifications into the workspace.
4. Click **Generate Tickets** and instantly copy or push your new tickets directly into production tracking.
